Contract Monthly Claim System (CMCS)
------------------------------------


Project Overview:
The Contract Monthly Claim System (CMCS) is a comprehensive web application developed for PROG6212 Programming 2B. This system streamlines the monthly claim submission and approval process for Independent Contractor lecturers within educational institutions.
*********
Features:

User Roles & Functionality

Lecturers: Submit monthly claims with auto-calculation, upload supporting documents, and track claim status

Programme Coordinators: Review and approve/reject pending claims with automated validation

Academic Managers: Final approval authority with comprehensive claim oversight

HR Administrators: Generate reports, manage users, and process payments
*********

Core Features:

Real-time claim amount calculation

Secure document upload with validation

Role-based access control

Automated approval workflows

Comprehensive reporting and analytics

Responsive Bootstrap 5 interface

In-memory database for easy demonstration
*********
Technology Stack:

Backend: ASP.NET Core 9.0, C# 12.0

Frontend: Bootstrap 5.3, JavaScript, jQuery

Database: Entity Framework Core with In-Memory Provider

Authentication: ASP.NET Identity

Testing: xUnit

*********
Prerequisites:

.NET 9.0 SDK

Visual Studio 2022 or VS Code
*********
Installation:

Clone the repository

Open the solution in Visual Studio

Restore NuGet packages

Build the solution

Run the application
